VDungeon

2021 Autumn
KTH course DH2413 Advanced Graphics and Interaction Project

  • View VDungeon Project Website

This game provides users a virtual way to play RPGs (e.g. D&D) with others remotely, which uses webcams to track players' facial expressions and project them to a virtual avatar. Several tabletop RPG mechanics (such as a board editor, interactive properties, a dice rolling system) are also supported in VDungeon.

Contribution

This is a group project cooperated with 4 other KTH students. In this project, I mainly contributed to the control of the game, UI design and implementation, particle effects as well as simple global illumination.
