FlightWise

2020 Spring
KTH course DH2321 Information Visualization Final Project

  • Visit FlightWise

This is a data visualization system showing carbon footprints of KTH fliers from 2017 to 2019. In this system, KTH flight carbon emissions change through the years, the overall/per capita emissions per school/department per year, and individual emissions are visualized to help KTH's goal of flight reduction.

Contribution

This is a group project cooperated with 4 other KTH students. In this group, I mainly contributed to frontend programming using D3.js and data analysis using Python.
